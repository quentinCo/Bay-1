# Bay-1
Projet IMAC2 - Visite virtuelle


Contrôles de navigation :

	z : aller en avant 
	s : aller en arrière
	q : aller à gauche
	d : aller à droite

	espace : monter
	shift : descendre

	left : aller au site précédent
	right : aller au site suivant
	
	f : full screan
